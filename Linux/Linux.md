# Linux常用指令

### yum

* install
  向系统中安装一个或多个软件包
* update
  更新系统中的一个或多个软件包

# 基本工具

### net-tools

#### 安装方法

`yum install net-tools -y`

#### 使用方法

##### route

查看当前网卡ip

* -h
  帮助信息
* -n
  查看网卡ip

##### ifconfig

查看网卡

* -h
  帮助信息
* -a
  查看网卡所有ip
